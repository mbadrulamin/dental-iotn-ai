# Frontend - Dental IOTN AI Platform

Next.js frontend for the Dental AI Research & Diagnostic Platform.

## Quick Start

```bash
# Install dependencies
npm install

# Set up environment
copy .env.example .env.local
# Edit .env.local with your API URL

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- 🦷 Public diagnostic tool with image upload
- 📊 IOTN DHC grade calculation
- 👨‍⚕️ Expert blind review interface
- 📈 Analytics dashboard with metrics
- 🔐 JWT authentication
